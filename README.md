# Datasets-for-Multimodal-Finger-Database.

This repository hosts a multimodal biometric dataset containing paired **Fingerprint (FP)** and **Finger Vein (FV)** images. 
## Dataset Structure
The dataset is organized into two main directories corresponding to the two biometric modalities:

* **`samples_fp/`**: Contains all fingerprint (FP) image samples.
* **`samples_fv/`**: Contains all corresponding finger vein (FV) image samples.
